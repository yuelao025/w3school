ASP.NET 教程

从左侧的菜单选择你需要的教程！

ASP.NET 是一个开发框架，用于通过 HTML、CSS、JavaScript 以及服务器脚本来构建网页和网站。

ASP.NET 支持三种开发模式：

  • Web Pages
  • MVC
  • Web Forms

进一步了解 ASP.NET 开发框架 ！

Web Pages

最简单的 ASP.NET 模型。

类似 PHP 和 ASP。

内置了用于数据库、视频、社交媒体等的模板和帮助器。

开始学习 ASP.NET Web Pages ！

Razor

Razor 是一种允许您向网页中嵌入基于服务器的代码（Visual Basic 和 C#）的标记语法。

Razor 基于 ASP.NET，它为 web 应用程序的创建而设计。它拥有传统 ASP.NET 标记的能力，但更易使用，也更易学习。

开始学习 Razor ！

MVC

MVC 将 web 应用程序分为三种不同的组件：

  • 针对数据的模型
  • 针对现实的视图
  • 针对输入的控制器

开始学习 ASP.NET MVC ！

Web Forms

传统的 ASP.NET 事件驱动开发模型。

添加了服务器控件、服务器事件以及服务器代码的网页。

开始学习 ASP.NET Web Forms ！
