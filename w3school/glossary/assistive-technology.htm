W3C 术语 - assistive technology

W3C 术语表中以字母 A 开头的条目

assistive technology

中文：辅助技术

解释

在本文档的语境中，辅助技术是这样的用户代理：依赖于由一个或多个其他“主”用户代理提供的服务（比如取回 Web 资源并解析标记）。辅助技术通过使用并监测 API 与主用户代理交流数据和消息。提供强于由主用户代理提供的服务，以满足残障人士的需求。附加的服务包括了替代呈现（例子，合成语言或放大的内容），替代输入方法（比如声音），替代导航或定位机制，以及内容转化（例如，使表格更具访问性）。在本文档的语境中重要的辅助技术实例包括以下：屏幕放大器，有视觉残疾的用户可以用其放大并改变屏幕上的颜色，以此改善所呈现文本和图像的视觉可读性；屏幕阅读器，盲人或有阅读障碍的用户可通过合成语音或盲文电子法来用其阅读文本信息；有某种生理残疾的人士可使用的声音识别软件；有某种生理残疾的人士可使用其模拟键盘的替代键盘；有某种生理残疾的人士可使用其模拟鼠标定位和按钮动作的替代定位设备。本文档的范畴之外，辅助技术由软件和硬件组成，它们为残疾人进行了特殊设计，旨在帮助他们完成日常活动。这些技术包括轮椅、阅读器、抓握设备、文字电话以及震动寻呼机。例如，下面是来自 (U.S.) Assistive Technology Act of 1998 [AT1998] 中有关“辅助技术设备”的非常通用的定义：设备或生产系统的任何项目或部分，无论是否是已获利为目的后天的，改良的，或定制的，用于提升、维持或改善残疾个人的功能性能力。

来源

来自：User Agent Accessibility Guidelines 1.0 (2002-12-17)

该来源中的术语

assistive technology

中文：辅助技术

解释

为残疾人进行了特殊设计旨在帮助他们完成日常活动的软件和硬件。辅助技术包括轮椅、阅读器、抓握设备等。在 Web 可访问性领域，普通的基于软件的辅助技术包括屏幕阅读器、屏幕放大器、语音合成器以及与图形桌面浏览器（在其他用户代理当中）结合的声音输入软件。硬件辅助技术包括替代键盘和定位设备。

来源

来自：Web Content Accessibility Guidelines 1.0 (1999-05-05)
