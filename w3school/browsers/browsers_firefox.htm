Mozilla Firefox 浏览器

  • Mozilla
  • Opera

Firefox 是由 Mozilla 开发的开源 web 浏览器。

Firefox 是什么？

Firefox 是一款免费的可用于 Windows、Linux 和 Mac 的开源 web 浏览器。它小巧快速，可提供弹窗阻止和标签式浏览等许多新特性。

Firefox 基于 Mozilla 代码，是标准兼容性最好的浏览器之一。

Firefox 可用于 Windows 98、Windows 2000、Windows XP、Mac OS X 以及 Linux。

Firefox 3

Mozilla 的最新版浏览器，Firefox 3，于 2008 年 6 月 17 日发布了。

Firefox 3 拥有众多新特性，比如密码管理、一键保存书签、智能地址栏、全屏缩放、快速网站 ID 信息、平台原生外观等等。

对于 Mozilla 来说一项很重要的特性，是对性能的提升。Firefox 3 可以使您更快地浏览网页，同时占用更少的计算机内存资源。

Mozilla 是什么？

Mozilla 是一个开源的 web 开发项目。

Mozilla 应用程序套件（也称为 SeaMonkey）是一套完整的 web 应用程序；浏览器、电子邮件客户端、新闻客户端、聊天客户端等等。

而 Firefox 仅仅是一款浏览器。您无法使用 Firefox 来聊天或阅读电邮或新闻组。这使它更小、更快，更易使用（但却使得聊天、电邮以及新闻组更难使用？）

提示：您可以在上一节阅读更多有关 Mozilla 的内容。

Firefox 和 Mozilla

在过去 Firefox 被称为 Mozilla Firebird（而 Mozilla Firebird 过去被称为 Phoenix，Phoenix 过去被称为 Mozilla）。

要解决名称混乱的问题，就必须建立合法的发行号（我这么认为）。您可以在上一节阅读更多有关 Mozilla 演变改变的内容。

Firefox预期会成为完整版 Mozilla 套件的默认浏览器。

Firefox 的特性

Firefox 可自动进行弹窗阻止。这一点用于对付讨厌的弹出广告是很有效的。

标签式的浏览是一种现代的省时的的特性。它可以做到在同一个浏览器窗口查看多个页面。

Google Search 也被嵌入 Firefox 的工具栏中。

Keywords for bookmarks with quick searches (e.g., type "goo xml tutorial" into the location bar and get a Google search page on "xml tutorial" in one step).

词典工具提示 - 双击任意单词就会看到其定义。

Firefox增强了安全性能。它没有与操作系统整合，不支持 ActiveX 控件和 VBScripts （以安全漏洞闻名的特性）。

内置的现代下载管理器可在后台把文件下载到桌面。

Firefox 拥有可定制的工具栏，允许用户添加或删除项目，也可创建新的工具栏。

通过主题管理器，用户可以改变 Firefox 的外观。

Firefox 可自我更新。智能更新可在后台进行搜索，并通知用户可用的更新。
