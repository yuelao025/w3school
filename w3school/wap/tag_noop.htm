WML <noop> 标签

WML 参考手册

定义和用法

<noop> 任务表示不进行任何动作（noop 指的是 "no operation"）。此标签用于防止事件的发生。

注释：<noop> 可以位于 <do> 或 <onevent> 标签内。

语法

<noop/>

属性

属性 值 描述
class cdata 为此元素设置的 class 名。这个 class 名对大小写敏感。一个元素可链接多个 class。claas 属性中的多个 class 名通过空格进行分隔。
id id 为此元素设置的唯一名称。

实例

下面的例子使用 <do> 标签向卡片添加一个 "Back" 链接。当用户点击这个 "Back" 链接时，会跳转到前一个卡片。但是 <noop> 标签会阻止此操作；当点击 "Back" 链接时什么也不会发生：

<?xml version="1.0"?>
<!DOCTYPE wml PUBLIC "-//WAPFORUM//DTD WML 1.1//EN"
"http://www.wapforum.org/DTD/wml_1.1.xml">

<wml>
<card>
<p>
 <do type="prev" label="Back">
  <noop/>
 </do>
</p>
</card> 
</wml>
