floor() 函数返回不大于 x 参数值的最接近的整数。

语法

n = Float.floor(x)
成分 描述
n 该函数返回的整数。
x 一个数。

例子

var a = Float.floor(5.32);
var b = Float.floor(-5.32);

结果

a = 5
b = -6
