CSS3 :enabled 选择器

CSS 选择器参考手册

实例

为所有 type="text" 的已启用的 input 元素设置背景色：

input[type="text"]:enabled
{ 
background-color: #ff0000;
}

亲自试一试

定义和用法

:enabled 选择器匹配每个已启用的元素（大多用在表单元素上）。

浏览器支持

所有主流浏览器均支持 :enabled 选择器，除了 IE8 及更早的版本。

亲自试一试 - 实例

input:enabled
