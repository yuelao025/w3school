CSS [attribute=value] 选择器

CSS 选择器参考手册

实例

为 target="_blank" 的 <a> 元素设置样式：

a[target=_blank]
{ 
background-color:yellow;
}

亲自试一试

浏览器支持

所有主流浏览器都支持 [attribute=value] 选择器。

注释：对于 IE8 及更早版本的浏览器中的 [attribute=value]，必须声明 <!DOCTYPE>。

定义和用法

[attribute=value] 选择器用于选取带有指定属性和值的元素。
