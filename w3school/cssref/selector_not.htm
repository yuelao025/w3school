CSS3 :not 选择器

CSS 选择器参考手册

实例

设置非 <p> 元素的所有元素的背景色：

:not(p)
{ 
background-color: #ff0000;
}

亲自试一试

定义和用法

:not(selector) 选择器匹配非指定元素/选择器的每个元素。

浏览器支持

所有主流浏览器均支持 :not 选择器，除了 IE8 及更早的版本。
