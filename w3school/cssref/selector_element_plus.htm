CSS element+element 选择器

CSS 选择器参考手册

实例

选择 <div> 元素之后紧跟的每个 <p> 元素，并设置其背景色：

div+p
{ 
background-color:yellow;
}

亲自试一试

浏览器支持

所有主流浏览器都支持 element+element 选择器。

注释：对于 IE8 及更早版本的浏览器中的 element+element，必须声明 <!DOCTYPE>。

定义和用法

element+element 选择器用于选取第一个指定的元素之后（不是内部）紧跟的元素。
