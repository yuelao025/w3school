CSS :before 伪元素

CSS 参考手册

定义和用法

:before 伪元素在元素之前添加内容。

这个伪元素允许创作人员在元素内容的最前面插入生成内容。默认地，这个伪元素是行内元素，不过可以使用属性 display 改变这一点。

实例

此样式会在每个 h1 元素之前播放一段声音：

h1:before
  {
  content:url(beep.wav);
  }

TIY

浏览器支持

所有主流浏览器都支持 :before 伪元素。
